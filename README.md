# OCaml Template

This is a template for OCaml projects.

## Quick Start

1. Fork this repository
2. Rename the repository to your project name
3. Replace all instances of `ocaml_template` with your project name, both in files and filenames themselves.

## Project Structure

```txt
.
├── _build                  # build outputs
├── bin                     # program entry points
├── docs                    # documentation
├── lib                     # public modules
├── test                    # tests
├── .editorconfig
├── .envrc
├── .gitignore
├── .ocamlformat            # ocamlformat configuration
├── CODEOWNERS
├── CONTRIBUTING.md
├── LICENSE
├── README.md
├── Taskfile.yml            # task runner configuration similar to make
├── dune-project	        # dune project configuration
├── flake.lock
├── flake.nix               # nix flake configuration
├── ocaml_template.install  # autogenerated install script
└── ocaml_template.opam     # autogenerated project description
```
